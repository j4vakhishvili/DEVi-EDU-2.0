სათაური

HTML ატრიბუტები იძლევა დამატებით ინფორმაციას HTML ელემენტების შესახებ.

HTML ატრიბუტები

  • ყველა HTML შესაძლოა ჰქონდეს ატრიბუტები
  • ატრიბუტები იძლევა დამატებით ინფორმაციას ელემენტების შესახებ
  • ატრიბუტები ყოველთვის მითითებულია საწყის ტეგში
  • ატრიბუტები როგორც წესი იწარმოება მსგავსად: სახელი="მნიშვნნელობა"

ატრიბუტი - href

<a> თეგი აწარმოებს ლინკებს. href ატრიბუტი წარმოადგენს URL-ს რომელიც გადადის მითითებულ ლინკზე.

ლინკებზე მეტის გასაგებად ეწვიეთ ამ გვერდს

ატრიბუტი - src

<img> თეგი გამოიყენება ფოტოების HTML გვერდზე გამოსატანად. src ატრიბუტი აწვდის ადგილის რომელიც მითითებულია სურათის გამოსატანად.

მხოლოდ ორი გზაა, რათა ლინკების მიწოდება მოხდეს src ატრიბუტის მეშვეობით:
1. ბმულის სახით - ლინკების მეშვეობით, რომლებიც მოწოდებულია რომელიმე სხვა ვებსაიტიდან. (მაგ.: src="https://www.edu-devi.tk/image.png/")
შენიშვნა: საიტის გარედან შემოტანილი ფოტოები შესაძლოა შეიცავდეს საავტორო უფლებებს, თუ თქვენ არ ფლობს საავტორო უფლებებს შესაძლოა შეგექმნათ კანონთან არსებული პრობლემები. ამავდროულად გაითვალისწინეთ, რომ თქვენ არ შეგიძლიათ მათი კონტროლი, არსებული ფოტოები შესაძლოა მათი სერვერებიდან წაიშალოს ან შეიცვალოს.
2. შიდა ფაილის სახით - მსგავის ტიპის სურათები საიტთან ერთად გაეშვება ონლაინში, შესაბამისად სურათს არ სჭირდება საიტის მისამართი, იგი პირდაპირ თქვენი ფაილებიდან ინლინკება. მაგალითად: src="image.png", თუ კი ფოტო სხვა ფაილში გაქვთ ამისთვის დაგჭირდებათ ფაილის არჩევა. მაგალითად: src="assets/image.png"
შენიშვნა: საუკეთესო ვარიანტია როდესაც ფოტოების საიტზე განთავსება შიდა ფაილის სახით ხდება, ვინაიდან მსგავს სიტუაციაში, ფოტო არასდროს წაიშლება, მაშინაც კი, თუ საიტი შეიცვლის მისამართს.

სიგრძის და სიმაღლის ატრიბუტები

<img> თეგი ასევე შესაძლოა შეიცავდეს width და height ატრიბუტებს, რომლებიც სურათს ანიჭებენ სიგრძეს და სიმაღლეს (პიქსელებში).


ატრიბუტი - alt

<img> თეგისთვის alt ატრიბუტი გამოიყენება მაშინ როდესაც ფოტო რაღაც მიზეზების გამო არ გამოისახება, ეს შესაძლოა გამოწვეული იყოს დაბალი ინტერნეტის კავშირის გამო ან არასწორად მითითებული ფაილის ლოკაციიდან გამომდინარე.

სურათებზე მეტის გასაგებად ეწვიეთ ამ გვერდს

ატრიბუტი - style

style ატრიბუტი გამოიყენება ელემენტისთვის სტილის მისანიჭებლად, როგორიცაა ფერი, ზომა, ფონტი ან სხვა.

სტილებზე მეტის გასაგებად ეწვიეთ ამ გვერდს

ატრიბუტი - lang

უმჯობესია ყოველთვის გამოიყენო lang ატრიბუტი <html> თეგში, ეს ეხმარება საძიებო სისტემას რეგიონის სწორად შერჩევაში.


ატრიბუტი - title

title ელემენტი ინახავს დამხმარე ინფორმაციას ელემენტზე, მისი ნახვა შესაძლებელია მასთან მაუსის მიტანით.

ეს პარაგრაფია.


შეჯამება

  • src ატრიბუტები <img> ტეგში წარმოადგენს ფოტოს ადგილმდებარეობის მითითებას
  • width და height ატრიბუტები <img> ტეგში წარმოადგენს ფოტოს ზომის მითითებებს
  • alt ატრიბუტი <img> ტეგში გამოიყენება მაშინ როდესაც ფოტო რაღაც მიზეზების გამო არ გამოისახება
  • style ატრიბუტები გამოიყენება ელემენტისთვის ფერის, ფონტის და მსგავსი ტიპის საკითხების შესაცვლელად
  • lang ატრიბუტები <html> ტეგში წარმოადგენს ინფორმაციას ვებ გვერდის ენის შესახებ
  • title ატრიბუტები წარმოადგენს დამატებითი ინფორმაციას მიმთითებელს ატრიბუტისთვის
