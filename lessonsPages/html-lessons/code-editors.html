სათაური

მარტივი კოდედიტორი - სულ ესაა რაც გჭირდება.

ვებ საიტის დაწერა შენ ძალიან მარტივად, ტექსტ ედიტორშიც კი შეგიძლია
HTML-ს საწყის დონეზე შესასწავლად შენ შეგიძლია გამოიყენო Notepad (Windows) ან TextEdit (Mac).

ნაბიჯი I
(Windows)

გახსენით "სტარტი" (დააკლიკეთ ვინდოუსის ლოგოს), მოიძიეთ Notepad სტარტის საძიებო სისტემაში.

(MAC)

გახსენით "Finder" შემდგომ > Applications > TextEdit
ასევე შეცვალეთ რამდენიმე ოფცია, სწორად შესანახად. Preferences > Format > აირჩიეთ "Plain Text"
შემდეგ კი მონიშნეთ ღილაკი რომელსაც აწერია "Display HTML files as HTML code instead of formatted text".
საბოლოოდ გახსენით ახალი დოკუმენტი კოდის დასაწერად.

ნაბიჯი III

შეინახე სურათზე მითითებული პარამეტრების მიხედვით.

ფაილის შენახვა შესაძლებელია როგორც .html დაბოლოებით ასევე .htm

ხილული სხვაობა ამ ორს შორის არ არის, უბრალოდ .htm ძირითადად ოდნავ ძველი ვერსიის html-ს მოიცავს

საბოლოო ჯამში არჩევანი შენზეა, მაგრამ ხშირად გამოყენებადი მაინც .html ტიპის დაბოლოებაა.

..თუ კი შენ ეძებ მეტად კომფორტულ კოდედიტორებს, ვიდრე სისტემის ჩაშენებული ტექსტედიტორებია, მაშინ გირჩევდი Visual Studio Code-ს ან მის ონლაინ ვერსიას.

აქვე დავამატებდი, რომ ჩვენ გვაქვს საკუთარი კოდედიტორი, რომელიც შეგიძლიათ გამოიყენოთ უეცარი საჭიროებისას.
