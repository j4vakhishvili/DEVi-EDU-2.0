სათაური

HTML-ში ფერები გამოიხატება წინასწარ შედგენილი სახელებით, ან RGB, HEX, HSL და სხვა მრავალი კოდის მეშვეობით.

ფერთა სახელები

HTML-ში ფერების გამოყენება შესაძლებელია მათი სახელების მეშვეობით:

Tomato Orange Plum RebeccaPurple OliveDrab

HTML მხარს უჭერს 140 სახეობის ფერის სახელს.

ფერები ყველგან

შენ შეგიძლია მიანიჭო ფერი ფონს, ტექსტს, ობიექტის საზღვრებს:


ფერის კოდები

HTML-ში, ფერები ასევე გამოიხატება კოდის სახით, როგორებიცაა RGB, RGBA, HEX, HSL, HSLA.
ქვემოთ ნაჩვენებ Div ელემენტებს ფონის ფერი აქვთ RGB, HEX და HSL კოდებში:

rgb(192, 3, 61)

#C0033D

hsl(342deg, 97%, 38%)

ქვემოთ ნაჩვენებ ელემენტებს ფონის ფერი აქვთ RGBA და HSLA კოდებში, რომლებსაც გამჭირვალობა აქვს დაწეული:

rgb(192, 3, 61, 0.5)

hsl(342deg, 97%, 38%, 0.5)

ფერები იწარმოება background-color, color, border-color მნიშვნელობებით.

RGB ფერები

RGB ფერები იწარმოება წითელი, მწვანე და ლურჯი ნათების მეშვეობით.

RGB ფერების შესაქმნელად გამოიყენება შემდეგი ფორმულა rgb(წითელი, ლურჯი, მწვანე) თითოეული პარამეტრი განსაზღვრავს ფერის ინტენსივობას 0 -დან 255 -მდე მნიშვნელობით. ეს კი იმას ნიშნავს, რომ 256 x 256 x 256 = 16777216 შესაძლო ფერს. მაგალითისთვის rgb(255, 0, 0) ფერი იქნება წითელი, ვინაიდან წითელი ფერის 255-ე მნიშვნელობაა აღწერილი ხოლო ლურჯი და მწვანე განულებულია. შავის მისაღებად მოგიწევთ ყველა პარამეტრის განულება, ხოლო თეთრისთვის თითოეული პარამეტრის მაქსიმალურ ღირებულებამდე გაზრდა. ნაცრისფერი ფერის მისაღებად კი მნიშვნელოვანია ყველა პარამეტრის გათანაბრება, მაგალითად rgb(120, 120, 120) ან rgb(180, 180, 180). ხოლო ფერისთვის გამჭირვალობის მისაცემად საჭიროა ალფა გამოსხივების დაწევა შესაბამისად rgba(255, 187, 0, 0.5).

HEX ფერები

HEX ფერები იწარმოება როგორც #RRGGBB, RR(წითელი) GG(მწვანე) BB(ლურჯი).

HSL ფერები

HSL ფერები იწარმოება როგორც ელფერი(hue), გაჯერება(saturation), და სინათლე(lightness).
