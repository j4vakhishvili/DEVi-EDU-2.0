სათაური

CSS იშიფრება როგორც კასკადური სტილური ფურცლები(Cascading Style Sheets).
CSS ზოგავს სამუშაოს, მას შეუძლია აკონტროლოს სტილები სხვადასხვა გვერდზე ერთდროულად.

რა არის CSS?

CSS-ს ვიყენებთ ვებსაიტისთვის სტილების მისაცემად.
CSS-ს მეშვეობით შენ შეგიძია აკონტროლო ობიექტის ფერები, ზომები, ფონტები, დაშორებები ელემენტებს შორის, პოზიციები და კიდევ უამრავი რამ!

შენიშვნა: სიტყვა კასკადური ნიშნავს რომ სტილი, რომელიც მინიჭებული აქვს მშობელ ელემენტს ასევე ენიჭება მის შიგნით განთავსებულს ობიექტს. ანუ თუ body-ს მივანიჭებთ სტილს color: blue; მის შიგნით განთავსებული ყოველი სათაური, პარაგრაფი თუ ტექსტი იქნება ლურჯი. (თუ სათითაოდ არ მიანიჭებ ელემენტს რაიმე განსხვავებულს)

CSS-ის გამოყენება

CSS-ს HTML დოკუმენტში დამატება სამი ტიპის ხერხით შეიძლება:

  • კონკრეტულ ხაზზე - style ატრიბუტის გამოყენებით HTML ელემენტში.
  • ჩაშენებული - <style> ელემენტის გამოყენებით <head> სექციაში.
  • გარე დოკუმენტში - <link> ელემენტის გამოყენებით <head> სექციაში გარე ფაილის მიბმა.

გაითვალისწინე, რომ ყველაზე მიღებული ფორმა CSS-ს მიბმისა არის გარე დოკუმეტის გამოყენება.

კონკრეტულ ხაზზე განთავსებული CSS

მსგავსი ტიპის CSS გამოიყენება იმისთვის რომ ელემენტს მივანიჭოთ უნიკალური სტილი, ისეთი რაც არ გამეორდება.

ნაჩვენებ მაგალითში ჩანს თუ როგორ მივანიჭოთ <h1>-ს ნარინჯისფერი.
