სათაური

HTML არის სტანდარტული მერქაფ ენა ვებსაიტების შესაქმნელად.

რა არის HTML?

  • HTML იშიფრება როგორც: Hyper Markup Language.
  • HTML არის ვებ-გვერდების შექმნისთვის განკუთვნილი სტანდარტული მარკირების ენა.
  • HTML აღწერს ვებ-გვერდის სტრუქტურას.
  • HTML მოიცავს ელემენტების მრავალ ჯგუფს.
  • HTML ელემენტები გადასცემენ ბრაუზერებს კონტენტის გამოსახვის საშუალებებს.

მარტივი HTML დოკუმენტი:



მაგალითის ახსნა:

  • <!DOCTYPE HTML> დეკლარაცია განსაზღვრავს რომ ეს დოკუმენტი არის HTML დოკუმენტი.
  • <html> ელემენტი არის HTML გვერდის საწყისი ელემენტი.
  • <head> ელემენტში ინახება ინფორმაციები, ვებ გვერდის შესახებ. (ენა, დისტრიბუცია, ავტორი, სათაური და ა.შ.)
  • <title> ელემენტი ინახავს ვებ-გვერდის სათაურს (რომელიც შემდგომ ნაჩვენებია ბრაუზერის სათაურის ბარში ან გვერდის ტაბში).
  • <body> ელემენტი განსაზღვარვს დოკუმენტის "სხეულს", აქ ჩანს დოკუმენტის მთავარი კონტენტი, მასში შესაძლოა ისეთი ატრიბუტების შეტანა როგორებიცაა ნავიგაცია, პარაგრაფები, ფოტოები, ლინკები, ცხრილები, სიები და ა.შ.
  • <h1> ელემენტი ძირითადად გამოიყენება სათაურის დასაწერად. (h1-ყველაზე დიდი ტექსტ ატრიბუტია)
  • <p> ელემენტი განსაზღვრავს პარაგრაფს.
